MUD (Multi-User Dungeon) Сервер
-------------------------------

**Версия:** 1.0.0

**Автор:** bronichev.a.r

**Дата:** 2026

Задание на разработку
--------------------

Реализация сервера для многопользовательской текстовой игры (MUD) со следующим функционалом:

1. **Игровое поле:** Двумерное закольцованное поле размером SIZE (ширина × высота)

2. **Игроки:** 
   - Подключение к серверу с уникальным именем
   - Перемещение по полю (вверх/вниз/влево/вправо)
   - Общение со всеми игроками (sayall)
   - Отключение от сервера (stop)

3. **Монстры:**
   - Добавление монстров на поле через команду addmon
   - Каждый монстр имеет: имя, приветственную фразу, здоровье
   - Встреча с монстром при входе на клетку (энкаунтер)
   - Система боя с монстрами (команда attack)

4. **Бродячие монстры:**
   - Каждые 30 секунд случайный монстр перемещается на одну клетку
   - Направление выбирается случайно (вправо/влево/вверх/вниз)
   - Если целевая клетка занята другим монстром - выбор повторяется
   - При перемещении всем игрокам отправляется уведомление
   - При попадании на клетку с игроком происходит энкаунтер

5. **Технические требования:**
   - Асинхронная архитектура на базе asyncio
   - Поддержка множества одновременных подключений
   - Корректная обработка отключений клиентов
